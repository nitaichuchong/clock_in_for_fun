# 这是什么？

---



一个整活用的小软件，给群友们开发的本质打卡类玩意儿，戒冲小助手

# 当前版本

---



1.0初始版

* 使用文件进行数据存储

* 统计戒冲打卡总天数和连续天数

* 每日只可打卡一次，通过系统时间进行判断

* pyqt制作的简陋ui
